### Day 17：魔术方法

#### 题目 1：实现常用魔术方法 ⭐⭐⭐
给 `Student` 类添加魔术方法：

```python
class Student:
    def __init__(self, name, scores):
        ...

    def __str__(self):
        # 打印对象时的输出
        return f"Student({self.name}, 平均分:{self.avg})"

    def __repr__(self):
        # 开发者友好的表示
        return f"Student(name='{self.name}', scores={self.scores})"

    def __len__(self):
        # 返回科目数量
        return len(self.scores)

    def __eq__(self, other):
        # 比较两个学生的平均分是否相同
        return self.avg == other.avg

    def __lt__(self, other):
        # 按平均分排序用
        return self.avg < other.avg
```

测试：
```python
s1 = Student("小明", [85, 90, 78])
s2 = Student("小红", [92, 88, 95])
print(s1)           # __str__
print(len(s1))      # __len__
print(s1 == s2)     # __eq__
students = [s2, s1]
students.sort()     # 用 __lt__ 排序
```

#### 题目 2：dataclass ⭐⭐⭐
用 `@dataclass` 重写 Student 类，对比和手写 `__init__` 的区别。

```python
from dataclasses import dataclass, field

@dataclass
class Student:
    name: str
    age: int
    scores: list[float] = field(default_factory=list)

    @property
    def average(self) -> float:
        return sum(self.scores) / len(self.scores) if self.scores else 0

# 测试 dataclass 自动生成的 __init__、__repr__、__eq__
```
