## 一、安装和环境准备

### 安装

```bash
# 创建虚拟环境（推荐）
python -m venv venv
# Windows 激活：
venv\Scripts\activate
# Mac/Linux 激活：
source venv/bin/activate

# 安装 FastAPI 和 uvicorn（服务器）
pip install fastapi uvicorn
```

### 第一个 FastAPI 应用

创建 `main.py`：

```python
from fastapi import FastAPI

app = FastAPI()

@app.get("/")
def read_root():
    return {"message": "Hello, World!"}
```

### 启动服务器

```bash
uvicorn main:app --reload
```

参数解释：
- `main` → 文件名（main.py）
- `app` → FastAPI 实例名
- `--reload` → 代码修改后自动重启（开发时用）

启动后访问：
- 你的 API：http://localhost:8000
- 自动文档：http://localhost:8000/docs ⭐
- 备用文档：http://localhost:8000/redoc

## 二、路由（Routing）

### 什么是路由？

路由 = URL 路径 + HTTP 方法 → 对应的处理函数

```python
@app.get("/")         # GET 请求访问根路径
@app.get("/users")    # GET 请求访问 /users
@app.post("/users")   # POST 请求访问 /users（和上面是不同路由！）
@app.put("/users/{user_id}")   # PUT 请求，带路径参数
```

### 同一路径不同方法

```python
@app.get("/items")
def list_items():
    return [{"id": 1}, {"id": 2}]

@app.post("/items")
def create_item():
    return {"id": 3, "message": "创建成功"}

# GET /items → 调用 list_items
# POST /items → 调用 create_item
```

## 三、路径参数（Path Parameters）

路径参数是 URL 路径中的**变量部分**：

```python
@app.get("/users/{user_id}")
def get_user(user_id: int):
    # user_id 会自动从 URL 中提取并转为 int
    return {"user_id": user_id}

# GET /users/123 → {"user_id": 123}
# GET /users/abc → 422 错误（不是整数）
```

### 类型验证是自动的！

```python
@app.get("/items/{item_id}")
def read_item(item_id: int):  # 声明为 int
    return {"item_id": item_id}

# 访问 /items/5 → 成功，item_id = 5
# 访问 /items/foo → 自动返回 422 错误（类型不匹配）
```

这就是 FastAPI 的核心魔法：**用 Python 类型注解自动做数据验证**。

### 枚举路径参数

```python
from enum import Enum

class ModelName(str, Enum):
    alexnet = "alexnet"
    resnet = "resnet"

@app.get("/models/{model_name}")
def get_model(model_name: ModelName):
    return {"model": model_name, "message": f"选择了 {model_name}"}

# GET /models/resnet → 成功
# GET /models/invalid → 422 错误（不在枚举中）
```

## 四、查询参数（Query Parameters）

查询参数是 URL 中 `?key=value` 部分：

```python
@app.get("/items")
def list_items(skip: int = 0, limit: int = 10):
    # skip 和 limit 是查询参数，有默认值
    return {"skip": skip, "limit": limit}

# GET /items → skip=0, limit=10
# GET /items?skip=5&limit=20 → skip=5, limit=20
```

### 路径参数 vs 查询参数

```python
@app.get("/users/{user_id}")
def get_user(user_id: int, include_details: bool = False):
    # user_id → 路径参数（必须在 URL 中）
    # include_details → 查询参数（可选，有默认值）
    return {"user_id": user_id, "details": include_details}

# GET /users/123 → user_id=123, include_details=False
# GET /users/123?include_details=true → user_id=123, include_details=True
```

**判断规则**：
- 参数在路径中 `{xxx}` → 路径参数
- 参数不在路径中 → 查询参数

### 必填查询参数

```python
@app.get("/search")
def search(q: str):  # 没有默认值 → 必填
    return {"query": q}

# GET /search → 422 错误（缺少 q）
# GET /search?q=python → 成功
```

### 可选查询参数

```python
from typing import Optional

@app.get("/search")
def search(q: str, page: Optional[int] = None):
    # page 是可选的，可以不传
    return {"query": q, "page": page}

# 在 Python 3.10+ 中可以写：page: int | None = None
```

## 五、返回值

### 返回字典（自动转为 JSON）

```python
@app.get("/user")
def get_user():
    return {"name": "张三", "age": 21}
    # FastAPI 自动设置 Content-Type: application/json
```

### 返回列表

```python
@app.get("/users")
def get_users():
    return [{"name": "张三"}, {"name": "李四"}]
```

### 返回状态码

```python
from fastapi import status

@app.post("/users", status_code=status.HTTP_201_CREATED)
def create_user():
    return {"message": "创建成功"}
```

### 设置响应头

```python
from fastapi import Response

@app.get("/data")
def get_data(response: Response):
    response.headers["X-Custom-Header"] = "hello"
    return {"data": "value"}
```

## 六、自动文档（核心亮点）

FastAPI 会**自动生成**交互式 API 文档。

### 使用方法

```python
from fastapi import FastAPI

app = FastAPI(
    title="我的待办事项 API",
    description="一个学习用的 API",
    version="1.0.0"
)

@app.get("/items", summary="获取所有项目", tags=["Items"])
def list_items():
    """
    获取项目列表。

    支持分页和过滤。
    """
    return []

@app.get("/items/{item_id}", summary="获取单个项目", tags=["Items"])
def get_item(item_id: int):
    """通过 ID 获取项目详情"""
    return {"item_id": item_id}

@app.post("/users", summary="创建用户", tags=["Users"])
def create_user(name: str):
    """创建新用户"""
    return {"name": name}
```

启动后访问 `http://localhost:8000/docs`：
- 每个接口都有描述
- 可以**直接在页面上测试**（点 "Try it out" → 填参数 → Execute）
- 参数类型、必填/选填都标得清清楚楚
