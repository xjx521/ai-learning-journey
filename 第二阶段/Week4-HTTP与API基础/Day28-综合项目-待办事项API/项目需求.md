## 项目概述

搭建一个**待办事项管理 API**，支持完整的 CRUD 操作、搜索、分页和数据持久化。

### 功能清单

| 接口 | 方法 | 路径 | 说明 |
|------|------|------|------|
| 创建待办 | POST | /todos | 创建新待办事项 |
| 获取列表 | GET | /todos | 支持关键词搜索 + 分页 |
| 获取详情 | GET | /todos/{id} | 按 ID 获取单条 |
| 更新待办 | PUT | /todos/{id} | 全量更新 |
| 部分更新 | PATCH | /todos/{id} | 只改指定字段 |
| 删除待办 | DELETE | /todos/{id} | 删除指定待办 |
| 统计信息 | GET | /todos/stats/summary | 完成/未完成数量统计 |

### 数据模型

```json
{
    "id": 1,
    "title": "学习 FastAPI",
    "description": "完成 Day 26-28 的学习",
    "completed": false,
    "created_at": "2026-07-20",
    "category": "学习"
}
```

### Step 2 🔲 实现 CRUD 接口

**需求**：用 Pydantic Model 定义数据，实现增删改查。

**Pydantic 模型**：
```python
from pydantic import BaseModel, Field
from datetime import date

class TodoCreate(BaseModel):
    title: str = Field(..., min_length=1, max_length=100)
    description: str | None = None
    category: str = "未分类"

class TodoResponse(BaseModel):
    id: int
    title: str
    description: str | None
    category: str
    completed: bool
    created_at: str

class TodoUpdate(BaseModel):
    title: str | None = None
    description: str | None = None
    category: str | None = None
    completed: bool | None = None
```

**需要实现的接口**：

#### 2.1 POST /todos（创建）
- 接收 TodoCreate 请求体
- 自动生成 ID（取当前最大 ID + 1）
- 自动设置 created_at 为今天日期
- 默认 completed = False
- 返回 201 状态码和 TodoResponse

#### 2.2 GET /todos（列表）
- 返回 todos_db 中所有待办事项

#### 2.3 GET /todos/{todo_id}（详情）
- 按 ID 查找，找不到返回 404

#### 2.4 PUT /todos/{todo_id}（全量更新）
- 找到待办 → 用请求体替换所有字段（ID 和 created_at 不变）
- 找不到返回 404

#### 2.5 DELETE /todos/{todo_id}（删除）
- 找到待办 → 删除 → 返回 204
- 找不到返回 404

**关键代码提示**：
```python
# 自动生成 ID
new_id = max([t["id"] for t in todos_db], default=0) + 1

# 查找待办
def find_todo(todo_id: int):
    for todo in todos_db:
        if todo["id"] == todo_id:
            return todo
    return None

# 404 处理
from fastapi import HTTPException
if not todo:
    raise HTTPException(status_code=404, detail="待办事项不存在")
```

**验证标准**：
- [ ] 在 /docs 中能创建待办
- [ ] 能查看所有待办列表
- [ ] 能查看单条详情
- [ ] 删除后再查看返回 404

### Step 3 🔲 搜索 + 分页 + PATCH

**需求**：给 GET /todos 加上搜索和分页功能，实现 PATCH 部分更新。

#### 3.1 GET /todos（升级：搜索 + 分页）

查询参数：
- `keyword: str | None = None` — 搜索标题或描述中包含关键词的
- `category: str | None = None` — 按分类过滤
- `completed: bool | None = None` — 按完成状态过滤
- `page: int = 1` — 页码（从 1 开始）
- `size: int = 10` — 每页数量

返回格式：
```json
{
    "data": [...],
    "pagination": {
        "page": 1,
        "size": 10,
        "total": 25,
        "total_pages": 3
    }
}
```

**关键代码提示**：
```python
import math

@app.get("/todos")
def list_todos(keyword: str | None = None, category: str | None = None,
               completed: bool | None = None, page: int = 1, size: int = 10):
    # 1. 过滤
    result = todos_db.copy()
    if keyword:
        result = [t for t in result if keyword.lower() in t["title"].lower()]
    # TODO: category 和 completed 过滤

    # 2. 分页
    total = len(result)
    start = (page - 1) * size
    end = start + size
    page_data = result[start:end]

    return {
        "data": page_data,
        "pagination": {
            "page": page,
            "size": size,
            "total": total,
            "total_pages": math.ceil(total / size) if size > 0 else 0
        }
    }
```

#### 3.2 PATCH /todos/{todo_id}（部分更新）

- 只更新请求体中实际传的字段
- 使用 `model_dump(exclude_unset=True)` 获取实际传的字段

```python
@app.patch("/todos/{todo_id}", response_model=TodoResponse)
def patch_todo(todo_id: int, updates: TodoUpdate):
    todo = find_todo(todo_id)
    if not todo:
        raise HTTPException(status_code=404, detail="待办事项不存在")

    update_data = updates.model_dump(exclude_unset=True)
    todo.update(update_data)
    return todo
```

**验证标准**：
- [ ] GET /todos?keyword=学习 只返回标题包含"学习"的
- [ ] GET /todos?completed=true 只返回已完成的
- [ ] GET /todos?page=1&size=2 正确分页
- [ ] PATCH /todos/1 Body: {"completed": true} 只改 completed 字段

### Step 4 🔲 JSON 文件持久化

**需求**：把数据保存到 JSON 文件，重启服务器数据不丢失。

**实现思路**：
```python
import json
import os

DATA_FILE = "todos.json"

def load_todos():
    """启动时从文件加载数据"""
    if os.path.exists(DATA_FILE):
        with open(DATA_FILE, "r", encoding="utf-8") as f:
            return json.load(f)
    return []

def save_todos():
    """每次修改后保存到文件"""
    with open(DATA_FILE, "w", encoding="utf-8") as f:
        json.dump(todos_db, f, ensure_ascii=False, indent=2)

# 启动时加载
todos_db = load_todos()
```

**需要做的改动**：
- 在 POST、PUT、PATCH、DELETE 接口中，每次修改后调用 `save_todos()`
- 应用启动时用 `load_todos()` 初始化

**验证标准**：
- [ ] 创建几条待办 → 重启服务器 → 数据还在
- [ ] 删除一条 → 重启 → 确实被删除了
- [ ] 打开 todos.json 文件，内容是格式化的 JSON
